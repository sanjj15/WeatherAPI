# 🌦 Weather Dashboard App

A React + Vite based Weather Dashboard that displays current and past weather information.

## 📌 Features
- Fetches real-time current weather data from the OpenWeather API.
- Loads past weather data from a local dataset file (testset.csv).
- Search weather by city name.
- Displays temperature, humidity, and weather conditions.
- Clean and responsive user interface.

## 🛠 Tech Stack
- React 18
- Vite 5
- Axios (API requests)
- XLSX (CSV/Excel handling)

## 📂 Data Sources
- Current Weather → OpenWeather API
- Past Weather → Local CSV dataset (testset.csv)

## 🚀 How to Run
1. Install dependencies using `npm install`
2. Start the development server using `npm run dev`
3. Open `http://localhost:5173` in your browser

## 📎 Notes
Past weather data is limited to the dataset provided in testset.csv.
API key is required for fetching live weather data.
